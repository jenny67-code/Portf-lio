Projeto JokenPô

O projeto é um site interativo do jogo Jokenpô, onde o usuário joga contra a máquina. Ao escolher entre pedra, papel ou tesoura, o sistema gera uma escolha aleatória para o computador, exibindo o resultado: vitória, derrota ou empate. O site tem visual simples e dinâmico, com resposta imediata ao clique. Foi desenvolvido com HTML, CSS e JavaScript.

Projeto Sorteio

O projeto é um site simples e funcional para sorteio de números aleatórios. O usuário define um intervalo (ex: de 1 a 100) e, ao clicar no botão de sorteio, o sistema gera um número aleatório dentro desse intervalo. A interface é clara, e fácil de usar. O site foi desenvolvido com HTML, CSS e JavaScript.
